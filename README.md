## Download 
```
$ git clone https://github.com/docker-wordpress-setup
```

## Navigate to source
```
$ cd docker-wordpress-setup
```

## Down volume data(has any big changes in db setup) 
```
$ docker-compose down --volumes
```

## Run with docker-compose
```
$ docker-compose up -d
```
